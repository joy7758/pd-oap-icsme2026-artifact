# Contributing

Useful contributions include:

- artifact rerun reports;
- documentation issue reports;
- fixture improvement suggestions;
- semantic failure class refinement suggestions;
- comparator feedback;
- reviewer quickstart improvements.

Do not submit real confidential data. Do not submit personal data. Do not submit real public-sector decision data unless it is already public, safe to reuse, and appropriate for the repository.

When reporting a rerun issue, include:

- operating system;
- Python version;
- command run;
- observed output;
- expected output;
- relevant report file if available.

The artifact preserves these non-claims: no legal compliance claim, no production deployment claim, no human-study result claim, no external validation claim, and no independent rerun claim.
