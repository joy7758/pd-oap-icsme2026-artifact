# Simulation Limitations

| Limitation | Mitigation | Remaining risk |
|---|---|---|
| No human participants | State that this is a deterministic simulation. | No actual human reviewer performance can be claimed. |
| Deterministic scoring | Make rules explicit and reproducible. | The model may miss cognitive factors. |
| Author-designed scoring model | Document rules and task contract. | Independent scoring-model review would strengthen confidence. |
| Synthetic fixtures | Use the existing controlled fixture and mutation corpus. | Real workflow variability is not measured. |
| No production workflow | Keep claims at representation-level reviewability. | Operational deployment remains unevaluated. |
| No legal judgment | Include explicit non-claims. | Legal sufficiency remains outside scope. |
| Does not measure cognitive load | Avoid user-study language. | Reviewer time and effort remain unknown. |
| Does not establish population-level performance | Report deterministic ratios only. | No statistical prevalence or performance inference is supported. |
