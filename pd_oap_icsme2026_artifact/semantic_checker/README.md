# v2.0 Semantic Checker

This package contains the local PD-OAP v2.0 semantic checker for the expanded controlled fixture benchmark.

The checker evaluates the 45 v2.0 fixtures against semantic rules that distinguish schema-shaped JSON from reviewable operation-accountability statements. It reports whether each fixture matches its declared expected result and expected primary failure class.

The checker does not prove legal compliance, production readiness, real public-sector approval, or external implementation validation. It is a local reference checker for the v2.0 evaluation substrate.
