# v2.0 Checker Scope And Limits

## Checks

The v2 semantic checker checks:

- policy basis presence
- dataset reference closure
- catalog reference closure
- requester role constraints
- purpose constraints
- reuse condition completeness
- reuse condition consistency
- handoff reference closure
- review trigger presence
- review evidence presence
- policy version consistency
- integrity digest consistency

## Limits

The checker does not check legal sufficiency, policy correctness in the real world, real public-sector approval, production deployment readiness, complete interoperability across all data-space systems, or external implementation evidence.

It is a local v2 reference checker used to evaluate controlled fixtures and generated mutations.
