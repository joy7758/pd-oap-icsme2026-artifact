# v2.0 Fixture Expansion

This package creates the expanded PD-OAP v2.0 controlled fixture benchmark.

It responds directly to the TOSEM evaluation criticism that the evaluation appeared insufficient, that Section 9 was very brief, and that the manuscript did not provide adequate empirical validation.

## Target

- 45 fixtures
- 3 operation families
- 12 semantic failure classes
- two invalid fixtures per failure class

## Scope

This is a controlled fixture benchmark, not a population-scale empirical benchmark. It creates synthetic, executable, bounded fixtures for representation and diagnostic evaluation.

No v1.x artifact files are modified. The v1.x 8-fixture set remains a historical artifact sanity baseline.
