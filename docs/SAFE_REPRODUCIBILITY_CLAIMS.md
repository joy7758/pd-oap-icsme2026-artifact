# Safe Reproducibility Claims

## Safe

- The repository includes a reproducibility-hardening release.
- The repository includes a public rerun-feedback issue.
- The repository includes an automated artifact smoke test.
- If applied and passing, the matrix workflow tests the artifact smoke command across Ubuntu, macOS, Windows, and Python 3.10, 3.11, and 3.12.

## Unsafe

- The artifact was validated by outside rerunners.
- The artifact has independent human rerun evidence.
- The artifact is ready for production deployment.
- The artifact demonstrates legal compliance.
- The artifact has human-study results.
